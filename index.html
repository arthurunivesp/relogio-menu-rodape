Explicação da Conversão Binária

A hora e os minutos são convertidos para binário da seguinte forma:

  • Hora: A hora atual é convertida para um número binário de 6 bits.
  • Minutos: Os minutos atuais são convertidos para um número binário de 6 bits.

Por exemplo, se a hora for 5, a conversão para binário será 000101. Se os minutos forem 9, a conversão para binário será 001001.
